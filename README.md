# Getting Started

---

## Activating Virtual Environment

```
sudo apt-get install python3-venv
python3 -m venv .venv
source .venv/bin/activate
```

## Installing Depencies

```
pip install --upgrade pip
pip install -r requirements.text
```

## Running

```
python3 src/main.py
```

## dotenv

```
REDIS_HOST=
REDIS_PORT=
REDIS_USERNAME=
REDIS_PASSWORD=
```
